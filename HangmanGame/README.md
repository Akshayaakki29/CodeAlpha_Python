# Hangman Game

## Description
A text-based Hangman game where the player guesses a hidden word one letter at a time.

## Features
- Random word selection
- Six attempts
- Colorful terminal interface
- Win/Lose messages

## Technologies Used
- Python
- Colorama

## How to Run
1. Install Python.
2. Install the required library:
   pip install colorama
3. Run:
   python hangman.py
