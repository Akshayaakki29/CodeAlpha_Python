# Stock Portfolio Tracker

## Description
A Python application to calculate the total investment value of a stock portfolio.

## Features
- Add multiple stocks
- Enter stock quantity
- Calculates total investment
- Displays portfolio in table format

## Technologies Used
- Python
- Colorama
- Tabulate

## How to Run
1. Install Python.
2. Install the required libraries:
   pip install colorama tabulate
3. Run:
   python stock_tracker.py
