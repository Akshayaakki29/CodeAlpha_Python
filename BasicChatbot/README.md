# Basic Chatbot

## Description
This is a simple rule-based chatbot developed using Python for the CodeAlpha Python Programming Internship.

## Features
- User-friendly chatbot
- Colorful terminal interface
- Typing effect
- Answers basic questions
- Displays date and time
- Exit using "bye"

## Technologies Used
- Python
- Colorama
- Pyfiglet

## How to Run
1. Install Python.
2. Install the required libraries:
   pip install colorama pyfiglet
3. Run:
   python chatbot.py
